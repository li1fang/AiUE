# Support

## Use the Right Channel

- Questions about how the platform works: GitHub Discussions
- Bugs and regressions: Issues
- Stable-surface design proposals: Discussions or ADR-first design threads
- Security-sensitive reports: private security contact path

## What To Include

- platform lane
- workspace shape
- command or schema involved
- expected result
- actual result
- relevant report paths or JSON excerpts

## Public Alpha Expectation

Support in Alpha is best-effort, but the goal is to make bug reports and collaboration reproducible rather than ad hoc.
