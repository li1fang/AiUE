# G2 Legacy Archive

This directory preserves the historical `G2` fixed-stage experiment.

`G2` is no longer part of the active AiUE line. It was superseded by the current validated path:

`V1 -> D1 -> D12 -> Q1 -> Q2 -> Q3`

The archived files are kept for reference only:

- the original PowerShell runner
- the original Python gate runner
- the original fixed-stage profile

These files are intentionally not updated as part of active development. New work should build on the active line and on the stabilized dual-host architecture described in:

- [ADR-0003](C:/AiUE/docs/adr/ADR-0003-active-line-and-dual-host-stabilization.md)
- [s1_stabilization_checkpoint.md](C:/AiUE/docs/checkpoints/s1_stabilization_checkpoint.md)
